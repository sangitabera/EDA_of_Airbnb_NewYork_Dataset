# 🏙️ Airbnb New York Data Analysis & SQL Insights

## 📌 Project Overview
This project focuses on **Exploratory Data Analysis (EDA)** and **SQL-based insights extraction** on the Airbnb New York City dataset. The goal is to uncover key trends in pricing, availability, location distribution, and customer behavior.

This project demonstrates strong skills in:
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- SQL-based Data Analysis
- Data-driven decision making


## 📊 Dataset Description
The dataset contains Airbnb listings in New York City with features such as:

- Listing ID
- Host ID & Host Name
- Neighbourhood Group & Area
- Room Type
- Price
- Minimum Nights
- Number of Reviews
- Availability


## 🔍 Exploratory Data Analysis (EDA)

### Key Analysis Performed:
- Handling missing values and data cleaning
- Distribution of listings across boroughs
- Price analysis by location and room type
- Relationship between price and number of reviews
- Availability trends across different areas

### 📈 Insights:
- Manhattan has the highest number of listings and prices
- Entire homes/apartments are significantly more expensive
- Budget listings are concentrated in Brooklyn
- Listings with higher reviews tend to have moderate pricing


## 🧠 SQL Analysis

SQL queries were used to extract meaningful insights such as:

- Top expensive neighbourhoods
- Average price by room type
- Most active hosts
- Listings availability trends
- Review-based popularity analysis
